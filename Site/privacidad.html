Privacidad

Política de privacidad.

Esta página explica qué datos puedo recibir a través de este sitio, para qué los uso y cómo puedes solicitar acceso, corrección o eliminación.

Última actualización: 8 de junio de 2026

Responsable

El responsable de este sitio es Adrián Deyme, Coach Ejecutivo ICF, radicado en Panamá.

Para cualquier consulta relacionada con privacidad o datos personales, puedes escribir a adrian@adriandeyme.com.

Datos que puedo recibir

Este sitio puede recoger datos en tres situaciones principales:

  • Cuando solicitas un recurso gratuito, como La Escucha Ejecutiva, puedo recibir tu correo electrónico.
  • Cuando agendas una conversación, Calendly puede recoger los datos necesarios para gestionar la reunión.
  • Cuando navegas por el sitio, herramientas de analítica pueden registrar información técnica y de uso, como páginas visitadas, dispositivo, navegador o eventos de conversión.

Para qué uso los datos

Uso los datos únicamente para fines relacionados con este sitio y mi práctica profesional:

  • Enviar el recurso que solicitaste.
  • Gestionar conversaciones exploratorias o reuniones agendadas.
  • Responder consultas enviadas por email.
  • Medir el funcionamiento del sitio y entender qué contenidos resultan útiles.
  • Enviar comunicaciones relacionadas con liderazgo, coaching ejecutivo u organizaciones, cuando corresponda y siempre con posibilidad de dejar de recibirlas.

Herramientas externas

Para operar el sitio puedo usar servicios de terceros. Entre ellos:

  • Brevo, para gestionar contactos y el envío de recursos por email.
  • Calendly, para facilitar la agenda de reuniones.
  • Google Tag Manager y Google Analytics, para medición y analítica del sitio.
  • Vercel, para alojamiento e infraestructura del sitio.

Estos proveedores pueden tratar datos de acuerdo con sus propias políticas de privacidad y condiciones de servicio.

Base del tratamiento

Cuando solicitas un recurso o agendas una conversación, el tratamiento se basa en tu solicitud expresa. La analítica del sitio se usa para mejorar la experiencia, medir el rendimiento y entender el interés en los contenidos.

Conservación

Conservo los datos durante el tiempo necesario para cumplir el propósito para el que fueron recibidos, atender solicitudes, mantener registros razonables de comunicación o cumplir obligaciones aplicables.

Tus derechos

Puedes solicitar acceso, corrección, actualización o eliminación de tus datos escribiendo a adrian@adriandeyme.com. También puedes pedir dejar de recibir comunicaciones cuando lo desees.

Cookies y medición

El sitio puede utilizar cookies o tecnologías similares para analítica, medición de conversiones y funcionamiento de herramientas integradas. Puedes administrar o bloquear cookies desde la configuración de tu navegador.

Cambios en esta política

Esta política puede actualizarse para reflejar cambios en el sitio, en las herramientas utilizadas o en requisitos aplicables. La fecha de actualización indica la versión vigente.

Nota

Esta política está redactada para explicar de forma clara el uso básico de datos en el sitio. Si en el futuro se agregan newsletters regulares, campañas pagas, formularios más extensos o nuevas integraciones, conviene revisarla nuevamente.
